# SendDB
Easily track level sends. View data from [SendDB](https://senddb.dev) in-game with this mod!

## Features
- <cg>Clean UI and charts</c> for data visualisation
- Level and creator <cy>statistics and breakdowns</c>
- <cg>Most sent level leaderboards</c> with filters
- Live <cp>trending levels</c> based on recent send activity
- <cy>Creator</c> <cg>send leaderboards</c> to track the most sent creators
